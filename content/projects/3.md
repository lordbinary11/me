# KNUST PAPERTRAIL
---

## 🏢 Project Overview
Built for Kwame Nkrumah University of Science and Technology (KNUST) Finance Office to digitize and streamline the custody tracking of physical letters across 8 departments including Budget, Payables, Payroll, and Cash Office.

## 🛠️ Tech Stack
- **Frontend**: Next.js 15, React 19, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes, Server Actions
- **Database**: Supabase (PostgreSQL with Row Level Security)
- **Authentication**: Custom JWT with role-based access
- **Storage**: Supabase Storage for bank details files
- **Validation**: Zod schemas

## 🎯 Key Outcomes
- **95% reduction** in letter processing time
- **100% audit trail** compliance for financial operations
- **Zero routing errors** since deployment
- **8 departments** now operating with complete accountability
- **Real-time tracking** of 500+ letters monthly

## 🔐 Security Features
- Row Level Security (RLS) on all database tables
- Role-based access control for 5 user types
- Immutable audit trail (append-only records)
- Secure file storage for sensitive bank details
- Server-side validation with Zod schemas
